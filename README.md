# Product Name & Price Scraper

## Project Description
This project scrapes product names and prices from a website and exports the data into a CSV file using Python.

## Data Collected
- Product Name
- Product Price

## Tools & Technologies
- Python
- Requests / Selenium
- BeautifulSoup
- CSV module

## Features
- Extracts product name and price
- Handles multiple products
- Saves data in CSV format

## Output
The scraped data is saved as:
- products.csv

## How to Run
1. Install required libraries
2. Run the Python script
3. CSV file will be generated automatically

## Note
This project is for educational purposes only.  
It follows ethical scraping practices and respects website policies.
